### What is Text Normalization ? 
+ Text Normalization is a fundamental text processing phase in Natural Language Processing (NLP).
+ It is the process of converting raw , messy , and varied text into a standard , unfiform and clean format 
** Example :**
+ In real-world text data , words can be written in multiple formats(e.g , Running, runs ,ran or RUNNING). Text Normalization reduces the variation , shrinking the vocabulary size so that Machine Learning and Deep Learning models can process the core meaning of the text more efficiently without getting confused
by superfical difference 

### Types of Text Normalization 
+ There are many types of text Normalization 
+ 1.Lowercasing 
+ 2.Removing punctuations & special characters and white space 
+ 3.Stemming 
+ 4.Lemmatization
+ 5.Exparding Contractions 
+ 6.Handling Numbers 
+ 7.Handling(URL, Email Address, and Hashtags)
+ 8.spellChecker
